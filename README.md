# VINYL

VINYL is a social music review platform scaffolded as a pnpm monorepo.

## Workspace

- `apps/web`: Next.js 14 web app
- `apps/mobile`: Expo SDK 51 mobile app
- `packages/shared`: shared TypeScript domain logic and Supabase helpers
- `supabase`: config, migrations, and edge functions

## Commands

```bash
pnpm install
pnpm build
pnpm lint
pnpm dev
```
